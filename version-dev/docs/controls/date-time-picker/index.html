Date time picker

Usage​

Properties​

Props​

Name
Description and specifications
dateFormat
Since 1.1.0
Date format string. See @link https://moment.github.io/luxon/#/formatting?id=table-of-tokens for all available tokens.
Attribute:
date-format
Type:
string
Default:
'yyyy/LL/dd'
eventDelimiter
Since 1.1.0
Default behavior of the done event is to join the two events (date and time) into one combined string output. This combination can be configured over the delimiter
Attribute:
event-delimiter
Type:
string
Default:
' - '
from
Since 1.1.0
Picker date. If the picker is in range mode this property is the start date. Format is based on `format`
Attribute:
from
Type:
string
maxDate
Since 1.1.0
The latest date that can be selected by the date picker. If not set there will be no restriction.
Attribute:
max-date
Type:
string
minDate
Since 1.1.0
The earliest date that can be selected by the date picker. If not set there will be no restriction.
Attribute:
min-date
Type:
string
range
Set range size
Attribute:
range
Type:
boolean
Default:
true
showHour
Show hour input
Attribute:
show-hour
Type:
boolean
Default:
false
showMinutes
Show minutes input
Attribute:
show-minutes
Type:
boolean
Default:
false
showSeconds
Show seconds input
Attribute:
show-seconds
Type:
boolean
Default:
false
showTimeReference
Since 1.1.0
Show time reference input Time reference is default aligned with @see {this.timeFormat}
Attribute:
show-time-reference
Type:
any
Default:
undefined
textSelectDate
Since 1.1.0
Text of date select button
Attribute:
text-select-date
Type:
string
Default:
'Done'
time
Since 1.1.0
Select time with format string
Attribute:
time
Type:
string
timeFormat
Since 1.1.0
Time format string. See @link https://moment.github.io/luxon/#/formatting?id=table-of-tokens for all available tokens.
Attribute:
time-format
Type:
string
Default:
'TT'
timeReference
Set time reference
Attribute:
time-reference
Type:
"AM" ｜ "PM"
to
Since 1.1.0
Picker date. If the picker is in range mode this property is the end date. If the picker is not in range mode leave this value `null` Format is based on `format`
Attribute:
to
Type:
string
Default:
null
